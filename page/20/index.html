redis

  • 概念：是非关系型数据库 NOSQL(redis,mangoDB,hbase…>)

  • 用途

    • 用作缓存

  • 数据结构

    • 存储的是键值对，key是字符串,value有五种结构 String map linkedlist set sorted

    • 增删改查

  • 持久化
    • RDB
      • 默认的方式，不需要配置，在一定时间内检测key的变化，然后持久化数据
    • AOF
      • 配置redis.windows.conf，可以每次操作以后进行持久化,appendonly 设置为yes
  • Java客户端：Jedis
  • Jedis连接池：JedisPool

            2. /xxx/xxx:多层路径，目录结构

字数统计字: 826 | 阅读时长分钟 ≈ 3

servlet

体系结构

  • GenericServlet:实现了其他的方法（包括空实现），只将service() 方法仍作为抽象方法，继承的时候需要实现该方法
  • HttpServlet: 继承自GenericServlet，可以方便的判断请求方式(即post和get方式)，对Http协议的封装，简化操作。

servlet相关配置

  • urlpartten:Servlet访问路径
    • 一个Servlet可以定义多个访问路径：@webServlet({“/d1”,”dd1”,”ddd1”})
    • 路径定义规则 ：
      • /xxx
      • /xxx/xxx/x :多层路径
      • *.do
