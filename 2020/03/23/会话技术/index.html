会话技术

  • 一次会话中包含多次请求和响应
  • 功能：在一次会话的范围内共享数据
  • 方式：
    • 客户端使用Cookie
    • 服务器端Session

cookie

  • 客户端会话技术，将数据保存到客户端

  • 使用

    • 创建 cookie绑定数据

      new Cookie(String name .String value)

    • 发送cookie对象

      response.addCookie(Cookie cookie)

    • 获取cookie

      Cookies[] request.getCookies()
